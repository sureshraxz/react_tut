step 2: Signing up for youtube api
---------------------------------
1. Navigate to console.developers.google.com
2. search for Api manager -> youtube data api v3
3. enable it 
4. menu -> credentials -> create credentials -> cpy the api key 
5. use that key in code 
6. Install dependency -> npm install --save youtube-api-search


step 3: search bar functionality
---------------------------------
1. User enters input 
    - we need an input field in search-bar component
    - we need to track down the entered data - for which we are going to use 'controlled class component'    
2. pass that input to the youtube-search-api
    - As we have youtube-search-api in index.js - we need to export search-bar.js and import it into index.js
3. display the output 
    - As we've dynamic input from search-bar component - we need to use 'controlled class component' for index.js
